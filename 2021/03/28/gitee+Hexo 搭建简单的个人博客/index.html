gitee+Hexo 搭建简单的个人博客

1、gitee

Gitee.com 是 OSCHINA.NET 推出的代码托管平台,支持 Git 和 SVN,提供免费的私有仓库托管。目前已有超过 500 万的开发者选择 Gitee。

2、hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

3、部署博客构建环境

  • 安装git

  • 安装 node.js

  • 部署Hexo

    Win+R-> ‘cmd’ 打开命令行

    先配置npm的安装源

    npm config set registry https://registry.npm.taobao.org

    然后再是安装hexo

4、构建个人博客

  • 在码云上创建仓库
  • 先注册一个账号，新建一个仓库

  • 在你的电脑上创建Hexo文件夹创建一个文件夹，然后到这个文件夹里右键Git Bash Here运行hexo init（这里需要等待一段时间)

  • 添加SSH公钥

    打开刚才创建的文件夹，右键Git GUI Here

    Help->Show SSH key->Generate SSH KEY 创建密钥

    复制公钥到

    设置->(左边列表)SSH公钥(粘贴公钥)

  • 本地预览博客

  • 更改博客主题

    默认使用的主题是landscape，在目录themes/landscape下，更改一个个人喜欢的主题，如hexo-theme-yilia。

    git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

    主题库：https://hexo.io/themes/

  • 在_config.yml中配置Git

    注意：冒号后面一定要有空格，否则不能正确识别。

  • 发布到Gitee

  • 配置Gitee Pages


    在/source/_posts目录下就可以写我们的博客

    每一次修改本地库都要打一遍hexo clean && hexo g && hexo d来将本地库同步到Gitee，并且在Gitee Pages更新一下。

    个人博客效果参考：https://qlr-0.gitee.io/myblog

    参考网址：https://www.jianshu.com/p/7cc1320bae6b https://blog.csdn.net/meifannao789456/article/details/81532924
