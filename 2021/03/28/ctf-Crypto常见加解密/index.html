凯撒密码

凯撒密码最早由古罗马军事统帅盖乌斯·尤利乌斯·凯撒在军队中用来传递加密信息，故称凯撒密码。这是一种位移加密方式，只对26个字母进行位移替换加密，规则简单，容易破解。

栅栏密码

①把将要传递的信息中的字符排成m行。

②再将下面一行字母排在上面一行的后边，从而形成一段密码。

③例如：

m=3

去掉空格： Thereisnoflag

培根密码

培根密码实际上就是一种替换密码，根据所给表一一对应转换即可加密解密 。它的特殊之处在于：可以通过不明显的特征来隐藏密码信息，比如大小写、正斜体等，只要两个不同的属性，密码即可隐藏。

其中a和b可以交换，或者换成其他字符。

如若加上一点其他规则

加密规则:

  • A-M=>a,也就是说A到M的所有字母都替换成a.

  • N-Z=>b,t也就是说N到Z的所有字母都替换成b.

  • A-M<=a,也就是说a可以替换成A-M中的任何一个字母

  • N- Z<=b,也就是说a可以替换成N-Z中的任何一个字母
    明文:

    s u c c e s s
    baaab baabb aaaba aaaba aabaa baaab baaab

    密文:
    Now is a good time to attend college. School work is a good teacherand a good builder of character. Every man should be a student andlearn all that there is about a subject.

这里是将每一个单词的首字母，按上述加密规则，转换成a或b。

猪圈密码

猪圈密码（Pigpen cipher)（英语： pigpen cipher)，亦称共济会密码（英语： masonic cipher）或共济会员密码（英语： Freemason’s cipher），是一种以格子为基础的简单替代式密码。即使使用符号，也不会影响密码分析，亦可用在其它替代式的方法。

维基利亚密码

维吉尼亚密码（又译维热纳尔密码）是使用一系列凯撒密码组成密码字母表的加密算法，属于多表密码的一种简单形式。

维基利亚密码是进行行列查找的

比如：

明文： Thereisnoflag（同样是去掉空格）

选择某一关键词并重复而得到密钥，key = noflag

密钥： noflagnoflag

密文： Gvjceofbtqlgt
